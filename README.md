ConnectaTel – Análisis de Segmentación y Estrategia
Este repositorio contiene el análisis de datos realizado para el proyecto ConnectaTel.

El dataset analizado incluye perfiles de usuarios de telefonía, permitiendo la identificación de patrones de consumo, tratamiento de anomalías y segmentación demográfica para la optimización de planes comerciales.

🧠 Objetivo del análisis
Limpieza de datos: Tratamiento de valores atípicos (outliers) utilizando el método IQR para asegurar la calidad de la base.

Segmentación: Clasificación de usuarios basada en su comportamiento de uso (Bajo/Medio/Alto) y etapa de vida (Joven/Adulto/Adulto Mayor).

Insights Estratégicos: Identificación de "Power Users" y generación de recomendaciones para la creación de ofertas integrales (bundles) personalizadas que mejoren la rentabilidad y satisfacción del cliente.
